Mine Home page Researched Home Page
The concept of the homepage design was taken from the Mi Nepal official site. There are different web componets used by Mi Nepal on their site. I had choosed to keep the image slider on homepage as of Mi Nepal. The image slider on Mi Nepal page was not automatic i.e it should be clicked to change the picture. So, I had used javascript to create image slider that changes picture on certain intervals automatically.
Mine product page Researched Product Page
The concept of the product page development was taken from QLX official website where product listing is done on tiles. So, I had choosed to list my products on grid by giving different CSS property like hover effect on the cover images of phone which provides better UI/UX interface.
Mine Blog page Researched Blog Page
The concept of the blog page development was taken from Vox.com official website. Different components like ordered list, unordered list, container, etc. is used to develop blog page. Images are also used to make blog page look better.
Mine Contact Us Page Researched Contact Us Page
The concept of the contact us form page development was taken from Midas official website. Items are inside wrapper container and javascript function to validate form is also used. Different CSS property and javascript function is used in contact form to make it functional.
